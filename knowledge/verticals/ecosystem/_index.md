# Ecosystem Vertical

## One-Line Summary
Ecosystem connects BitGo to external networks, assets, and protocols — enabling asset onboarding, staking, institutional DeFi, token management, and Go Accounts as the integrated utilization hub.

## Strategic Framing (H1 2026)

**Theme**: Shift from custody-led onboarding revenue → utilization-led monetization via Go Accounts as the default integrated hub.

The core flywheel: assets come in via custody → Go Accounts enables them to work (staking, DeFi, lending, trading) → yield drives retention → more AUC drives more revenue.
